First steps

Before you get started, you should add the HAProxy versions you want to use. By adding HAProxy versions to the HAProxyHQ, you will be presented with all the information the Configuration Manuel of the given versions provides, while creating configs for you HAProxy instances.

You can do this in the Admin area, by adding a new version and supplying the URL to the Configuration Manual of that version.

Agents

The Agent is the most element part of the HAProxyHQ. Each HAProxy instance you want to monitor, has to have an Agent running on it.

As soon as the Agent is running on the HAProxy instance, you can add the Agent to HAProxyHQ by clicking on Agents. After it's been created, you'll be able to simply download the Agent config file and won't need to go through the hassle of configuring it yourself.

Once you've got everything up and running, you'll be able to keep track of the health of your Agent and you're in for a trouble-free management of your HAProxy config!

So go ahead and create your first Agent!

Schemas

By creating a Schema, you can create a HAProxy config, which you can then apply to multiple Agents.

This way you won't have to write the same config over and over again!

You can even create new Schemas, based on existing ones. But go and check it out yourself...
